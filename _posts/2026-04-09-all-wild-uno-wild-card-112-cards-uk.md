## How to play: Quick start guide
If you’ve played UNO before, you’ll pick this up quickly, but the 112-card edition brings its own flavor to the table’s rhythm. Here’s a practical, no-nonsense guide to getting started fast.

### Setup
- Shuffle the deck thoroughly. You’ll want a big “start” pile since more cards equal more drama.
- Deal seven cards to each player. The rest becomes the draw pile.
- Flip the top card to start the discard pile. If it’s a Wild or action card, the rules can vary, so decide on housing rules before the first game starts to prevent “but that’s not how my sister plays” type disputes.

### Rules quick version
- Match the card by either number or color to play. For Wild cards, you can declare the color as you see fit (in many UK households, this is the moment the crowd erupts in a chorus of cheers or groans).
- If you can’t play any card, you must draw one card. If you draw a playable card, you can put it down immediately—and the chain continues.
- Standard UNO endgame rule: say “UNO” when you have one card left. Forget this and you’ll face a gentle penalty and a chorus of mockery.
- The extra wild variants add shortcuts or “hand swap” moments that can dramatically alter the table’s power dynamics.

### Household rules and variants
- House rule: no calling UNO until you actually have one card left. It prevents premature celebrations and keeps everyone honest.
- Hand-swap rounds: occasionally, a Wild Swap card allows you to exchange hands with another player. This can be chaos, but it’s the kind of chaos that keeps a party going.
- Quick-end variants: some groups like to implement a “one-card sprint” where you can end the game faster if a certain combination is played, turning a long night into a surprisingly short victory lap.
