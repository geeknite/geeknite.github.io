## Performance: speeds, real-world tests, and “can you actually edit 4K on this?”
### The advertised claim vs. reality game
Crucial markets the X10 as a fast portable SSD with USB 3.2 Gen 2 connectivity, offering impressive read/write numbers on paper. In real life, speeds depend on host hardware, file type, and the age of your computer’s USB controller. It’s a classic case of “the numbers are nice, but real-world usage is where the rubber meets the road.” In lab-ish conditions, expect around the 900–1050 MB/s reads and something in the 800–1000 MB/s writes range for large sequential transfers on a capable USB 3.2 Gen 2 host. For random I/O performance, you’ll see the drive carry its weight, but don’t expect PCIe-like latencies when you’re dealing with random small file operations.

### Synthetic benchmarks: how we measured
We ran a handful of consistent tests using well-known benchmarks, with a clean system and a consistent OS environment. Here’s a summary of what you might typically see:
- Sequential read: around 1000 MB/s
- Sequential write: around 950 MB/s
- Random 4K read: a few hundred thousand IOPS depending on alignment and block size
- Random 4K write: likewise a few hundred thousand IOPS under ideal conditions
These results are indicative, not gospel. If your USB bus is sharing bandwidth with a fast charger, or you’re on a machine with a subpar USB-C controller, you’ll see numbers drift. It’s the same as taking a sports car to the grocery store: you still arrive fast, but you might not hit top speed in traffic.

### Real-world file transfers: what actually matters
Copying 100GB of mixed media, large video files, and thousands of small documents? Expect a multi-GB file transfer to breeze along at roughly 1 GB per second on a good host. Smaller files, metadata-heavy directories, or embedded NTFS features might temper speeds a bit due to file-system overhead and OS caching. In practical terms, you’ll see snappy backups, quick media library migrations, and a noticeably speedier editing workflow when working with external drives for scratch space or active project folders.

### Gaming and game libraries
If you’re using the X10 to cache game assets or host a portable library, you’ll appreciate the fast load times compared to older USB 3.0 externals. It’s not a miracle; you’ll still deal with texture streaming and game engine bottlenecks, but loading screens can be noticeably shorter. We tested with a few 4K asset packs and a compact library; the X10 delivered predictable results—read heavy workloads benefited most from the throughput advantage.
