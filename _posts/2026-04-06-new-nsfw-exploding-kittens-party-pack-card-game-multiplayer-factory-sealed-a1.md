## Rules notes and recommended house rules

While the base gameplay is approachable, lovers of homebrew content can try some simple house rules to keep things fresh:

- One card per player rule: in a round, each player can only use one action that affects the draw pile, preventing long chains of sudden death. 
- Mood setting: at the start of the round, players declare a theme for the round, and all jokes must align with the theme. This keeps humor focused and reduces awkward moments for some players. 
- The mercy rule: if someone has had two exploding kitten near misses, allow a temporary peace treaty for one turn for that player to catch their breath. This keeps the table from spiraling too hard while preserving excitement. 

As with any NSFW package, it helps to have an agreed boundary on what is acceptable. The card set itself provides the framework for jokes; your group provides the tone. A little pre-game chat goes a long way toward preserving friendships and rivalries in equal measure.
