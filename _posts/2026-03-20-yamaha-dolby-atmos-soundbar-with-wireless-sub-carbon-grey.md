## Setup Tips and Tseudo-Troubleshooting
- Make sure you’re using HDMI ARC/eARC for the best audio bandwidth and lip-sync performance.
- If you notice dialogue sounds muffled, try a quick EQ bump on the midrange or run a room calibration in the MusicCast app.
- Place the sub slightly off-center from the TV to avoid bass nodes that cancel out in the seat area.
- Update firmware if available; performance tweaks sometimes come from the code elves.
- For streaming apps, ensure your network is stable; wireless noise can color the bass response over time.
