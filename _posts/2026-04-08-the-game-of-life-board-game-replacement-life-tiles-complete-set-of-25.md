## How to replace, organize, and store: a practical setup guide

### Replacement steps in 4 simple moves
1) Clear the table and gather the existing LIFE tiles. Separate any damaged tiles and set them aside for disposal or repurposing in a craft project—yes, your board game deserves a future, too.
2) Open the 25-tile replacement set and compare values to the board’s reference sheet. Lay them out in numerical order if you’re a neat freak or color-coded by event type if you’re an art minimalist.
3) Replace the old tiles one by one, shuffling the order to minimize future confusion. If an old tile is stubborn, consider marginal notes on the back with a pencil to designate its value.
4) Store the new tiles in the included tray or a dedicated small pouch. Label if you’re prepping for a convention or a family game night where you’ll host a tile swap.

### Sorting strategies for repeat players
- Color code by family events vs. career events.
- Use a tiny card with the tile's value on top so you can quickly reference whether you want to redraw or skip.
- Create a “rotating reserve” stack for tiles that feel too powerful or too weak in your group’s play style.

### Cleaning and maintenance
- Wipe down tiles with a microfibre cloth if they get dusty. Do not soak in water—these are not bath toys.
- If corners start to fray, consider a light sealant or corner shields to extend tile life. It’s not glamorous, but it saves countless rounds of tile-swapping frustration.
