## Compatibility and installation guide
### Compatibility checklist
- Laptop must support DDR5 SODIMM memory at 5600 MT/s or higher
- The motherboard must have two available memory slots, or you must clear a slot to install the kit
- BIOS should recognize commonly installed modules, but some laptops require a BIOS update to enable XMP/DOCP profile switching in memory
- Voltage compatibility around 1.1 V; higher voltage budgets could cause heat or stability issues on some ultrabooks
- Make sure the laptop has adequate cooling; memory runs warm when pushed in heavy workloads, and poor cooling can throttle memory bandwidth advantage

### How to install
1) Power down and unplug the laptop; remove the battery if you can do so safely. 2) Open the back panel to access the RAM slots. 3) If you have a single existing module in one slot, you can replace it with one of the new 32 GB modules or install both new modules into two slots if you have a two-slot laptop. 4) Align the notches and carefully press down until the modules click into place. 5) Replace the panel, power up, and enter BIOS to confirm that the memory is recognized at 5600 MT/s. 6) In Windows or macOS, run memory stress tests or use the built-in performance monitors to verify stability. If you see any POST issues, re-seat the modules or swap their order and reboot.

> A quick tip: some laptops require you to set the memory speed manually in BIOS if the default profile is limited to 4200 or 4800 MT/s. If you don’t want to tinker, enable the XMP/DOCP profile if your system supports it.

## Frequent questions
### Will this kit work in my laptop?
If your laptop supports DDR5 SODIMM and has two slots, there is a decent chance this kit will fit. Always check your manufacturer’s memory specifications and BIOS compatibility notes before buying. Most DDR5 laptops will offer a memory profile setting that you can enable to reach the rated speed. If your laptop boots fine at 5600 MT/s, enabling the profile is recommended; if not, you may need to run at the default speed until you find a stable profile for your hardware.

Check your warranty policy and the laptop manual before you upgrade.
