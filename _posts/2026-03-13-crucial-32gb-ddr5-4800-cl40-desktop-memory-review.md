### Compatibility testing checklist
- Enable XMP/EOCP/DOCP in BIOS to achieve advertised speeds.
- Verify motherboard BIOS is up to date.
- Confirm your power supply can handle a modern platform without sweating.
- Run a memory stress test after enabling XMP to ensure stability across all memory slots.
- Check for BIOS microcode updates that improve memory compatibility with DDR5 modules.

If you do all that and your system still benches like a sleepy goblin, you may need to adjust memory voltage slightly or tighten/loosen timings a touch. This is where patience and small steps can yield big stability dividends.
