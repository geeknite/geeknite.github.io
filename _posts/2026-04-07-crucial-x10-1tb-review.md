## Performance and real world numbers

The X10 is a USB C 3.2 device, which means that you should expect fast transfers when connected to a suitable host. Real world numbers vary with the type of data. Sequential read speeds can approach 1000 MB per second in ideal conditions, while sequential write speeds are slightly lower. Random IO performance is more important if you work with many small files, and that is where the real tests show the friction of a classic mechanical drive long since left behind.

## Practical advice for maximizing performance

- Always use a USB C 3.2 or higher port on your host to unlock full speeds.
- Use a high quality USB C to USB C cable for optimal data integrity.
- Ensure the drive is not in direct sunlight or enclosed in heat sources when running heavy workloads.
- Format to exFAT if you need cross platform portability, otherwise NTFS for Windows only and APFS for Mac only.
