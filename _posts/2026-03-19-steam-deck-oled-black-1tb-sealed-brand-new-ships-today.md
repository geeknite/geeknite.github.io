Let’s address the big whispered myth: burn-in. OLED displays can burn in if you leave static UI elements on the screen for extended periods. In a handheld that’s constantly on, this is a legitimate concern. Valve has implemented typical mitigations—screen savers, pixel shift, and software-level protections—but the honest truth is: if you routinely leave the same UI chrome visible for hours on end, you’ll want to vary your content and maybe avoid long-term static imagery in idle states. In practical terms, if you rotate games and bring up a different menu at least every so often, you’ll minimize burn-in risk. OLED displays tend to excel in color accuracy, but the brightness ceiling can be a factor when you’re out in the sun.
