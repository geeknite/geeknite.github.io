## Compatibility and installation notes

### Motherboard and BIOS support

Gen5 NVMe drives require a motherboard that supports PCIe Gen5. If your board is Gen4 only, the drive will still function but at Gen4 speeds. This is not a fault of the drive; it is a compatibility reality. If you are building a new system, consider either a Gen5 motherboard or a BIOS update that unlocks Gen5 support for the PCIe lanes. If you are upgrading an older system, you might want to check if your PCIe lanes are running in x4 mode without any lane splitting that could cap speeds. 

### Cooling considerations

As discussed earlier, Gen5 drives can run hot under sustained load. A simple aftermarket heatsink on your motherboard or a laptop cooling pad can protect you from potential thermal throttling. If you are stabilizing a desktop rig, ensure there is adequate airflow around the drive bay. For those using compact builds, you might want to monitor thermals with a temperature monitoring tool and adjust fan curves accordingly. 

### Installation steps quick recap

1) Power off and unplug the system. 2) Open the case and locate an available M.2 slot. 3) Insert the drive with the notch oriented correctly. 4) Secure with the mounting screw. 5) Boot into BIOS and confirm Gen5 PCIe is enabled for that slot. 6) Install your OS drive or clone your existing OS onto the new drive. 7) Initialize and format the drive in the operating system. 8) Install essential software and enjoy the speed relief.
